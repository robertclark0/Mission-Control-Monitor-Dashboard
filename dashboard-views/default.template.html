MCMD provides a simple platform to visualize mission critical data. Built on AngularJS and designed to scale.

Get Started

Click 'Add Monitors to Dashboard' above or click 'Monitors' on the bottom of the navigation bar. The Monitors window provides a list of available monitors that can be added to the dashboard. Add multiple monitors of the same type by giving them distinct names. Click 'Loop' to continuously loop through the monitor list.

Customize and Share

MCMD was designed to accommodate any type of data or API. Custom dashboards have three components: the data source, the widget functionality and the dashboard view. The modular widget design even allows for sharing widgets between dashboards.
